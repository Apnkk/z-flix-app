## Installation

### Option 1 : Setup (Recommandé)
1. Téléchargez `Z-FLIX-Setup-(version).exe`
2. Exécutez le fichier
3. Suivez l'installation
4. Lancez Z-FLIX depuis le menu démarrer ou le bureau

### Option 2 : Portable
1. Téléchargez `Z-FLIX-Portable-(version).exe`
2. Placez-le où vous voulez
3. Double-cliquez pour lancer

## ✨ Fonctionnalités

- 🎥 **Streaming HD** — Qualité optimale sans navigateur
- 🚀 **Rapide** — Interface native, chargement instantané
- 🚫 **Sans publicité** — Expérience fluide et sans interruption
- 🔄 **Mises à jour auto** — Restez à jour automatiquement
- 🎮 **Discord RPC** — Partagez ce que vous regardez
- 👥 **Watch Party** — Regardez à plusieurs
- 📱 **Multi-profil** — Profils séparés pour chaque utilisateur
